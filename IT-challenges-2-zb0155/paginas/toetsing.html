Eerste examenkans

De student breidt twee applicaties uit volgens opgegeven eisen. Wanneer de opdrachten niet voor de deadline wordt ingeleverd, mag de student zijn opdracht niet toelichten en krijgt hij/zij bijgevolg geen score.

Elke opdracht telt mee voor 50% van het eindtotaal. Tijdens de examenperiode licht de student zijn/haar project toe.

Per project geldt volgende onderverdeling:

  • 40% - Geschreven code van het project
  • 60% - Mondelinge toelichting van het project

Wie code overneemt of kopieert of laat kopiëren, pleegt plagiaat en dit zal voorgelegd worden aan de examencommissie.
