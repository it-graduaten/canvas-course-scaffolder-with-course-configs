In de cursus "Frontend frameworks" worden de belangrijkste concepten uit de React bibliotheek besproken. Je leert een Single Page Application te bouwen waarin de cruciale concepten zoals state en componenten verwerkt zitten. Na deze cursus kan je de aangeleerde concepten niet enkel gebruiken om een SPA te bouwen, maar kan je ook eenvoudig de overstap maken naar mobiele applicaties met React Native of full-stack applicaties met frameworks zoals Next.js of Remix.

De cursus begint met het opdelen van een UI in herbruikbare componenten. We bespreken verschillende manieren waarop deze componenten gestyled kunnen worden en welke voor en nadelen elke manier heeft. Daarna voegen we interactiviteit toe via state en bespreken we hoe context gebruikt kan worden om state te delen tussen verschillende componenten. Eens deze basisconcepten beheert zijn, maken we de overstap naar SPA's die uit verschillende pagina's bestaan en waarin informatie doorgeven wordt via de URL. Tenslotte bespreken we hoe we TanStack Query kunnen gebruiken om met server-side data uit te lezen en aan te passen.

Na de React lessen geven we een inleiding op het schrijven van end-2-end tests. De leerstof uit deze les is framework agnostisch en kan toegepast worden voor elk JavaScript of TypeScript framework en zelfs op een klassieke HTML/JS website.
