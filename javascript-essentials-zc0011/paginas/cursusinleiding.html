Welkom in de cursus JavaScript.

In dit opleidingonderdeel maak je kennis met JavaScript, de tweede belangrijke programmeertaal in onze opleiding. We beginnen eenvoudig met de basisstructuren die jullie reeds kennen uit Programming essentials en OO en bouwen geleidelijk aan op naar zaken die cruciaal zijn voor elke JavaScript developer. Zoals het manipuleren van de DOM en het gebruik van functionele array methodes, ... Daarnaast is er ook de nodige aandacht voor debugging.
