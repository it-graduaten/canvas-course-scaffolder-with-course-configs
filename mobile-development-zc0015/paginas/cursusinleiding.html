Welkom in de cursus Mobile Development waarin je mobiele applicaties bouwt voor Android en iOS. Je bouwt de applicaties met React en het Expo meta-framework, verder gebruiken je file-based routing om verschillende pagina's toe te voegen aan de applicatie.

De cursus begint met het bouwen van een eenvoudige webpagina met React Native en de core components die daarin aangeboden worden, vervolgens breiden we de applicatie uit met routing (Expo Router) en een UI-componenten library. Vervolgens breiden we de applicaties uit met native features zoals het gebruik van de camera en het filesystem. Tenslotte bespreken we hoe we Firebase kunnen gebruiken om authenticatie en een database toe te voegen.
