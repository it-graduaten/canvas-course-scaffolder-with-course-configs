Leerdoelen:

  1. Je schrijft een API waarmee data uitgelezen en aangepast kan worden in een database (relationeel of document).
  2. Je maakt gebruik van een JavaScript ORM tool om de communicatie tussen database en programma te implementeren.
  3. Je gebruikt sessions of JSON Web Tokens om authenticatie en autorisatie toe te voegen aan een API.
  4. Je gebruikt CORS om bepaalde onderdelen van een API af te schermen of toegankelijk te maken vanuit een browser.
  5. Je selecteert, op basis van gegeven vereisten, tussen een klassieke server, serverless of edge function om een API te ontwikkelen en gebruikt de gekozen techniek op een correcte manier.
  6. Je publiceert een API, edge en server functions via een geautomatiseerd proces naar een cloud service.
